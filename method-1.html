Протокол 001: Інструменти та матеріали
ПРАВИЛО БАЗИ: Якісний інструмент — це 50% успіху. Ніколи не економте на флюсі та припої!

1. Паяльник (TS21 / TS101)

Портативний, з точним регулюванням температури. Маст-хев для роботи з дрібними контактами.

2. Кусачки (Бокорізи)

Для рівного та точного обрізання дротів і пластикових стяжок у важкодоступних місцях.

3. Пінцет (Прямий та вигнутий)

Необхідний для надійного утримання гарячих дротів та дрібних SMD-компонентів під час пайки.

4. Гексагональні викрутки (шестигранні)

Основні розміри шестигранників (H1.5, H2.0, H2.5) для закручування гвинтів рами та моторів.

5. Знімач ізоляції (Стрипер)

Для швидкого та безпечного зачищення силіконових дротів без пошкодження внутрішніх жил.

6. Припій (з флюсом)

Свинцево-вмісний (наприклад, ПОС-61). Набагато краще плавиться і тримає вібрацію, ніж безсвинцевий.

7. Гель-флюс для електроніки

Якісний безвідмивний флюс. Забезпечує ідеальне розтікання припою та блискучу краплю.

8. Термоусадкові трубки

Різних діаметрів. Використовуються для надійної ізоляції всіх місць з'єднання та спайки.

9. Нейлонові стяжки

Також використовується для фіксації кабелів, маунтів, передавачів та приймачів.

10. Надфілі

Інструмент для обов'язкового затуплення гострих країв карбонової рами перед збіркою.

КРИТИЧНА ПОМИЛКА: НІКОЛИ не використовуйте активний (кислотний) паяльний жир або кислоту! Вона неминуче роз'їсть мідні контакти та знищить плату вашого дрона за кілька тижнів.
